#React OvoBoilerplate v2

This is the version of OvoBoilerplate uses React 16.4 with Redux. 
The application demonstrates how to use Redux.

### Running the application
`yarn s` or `yarn start`

### Folder structure


    src
        ├───actions
        ├───components
        ├───constants
        ├───containers
        ├───middleware
        ├───reducers
        └───store
